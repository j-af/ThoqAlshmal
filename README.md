# موقع مؤسسة ذوق الشمال للمقاولات

موقع عربي متجاوب وجاهز للنشر على GitHub Pages.

## النشر على GitHub Pages
1. أنشئ مستودعاً جديداً في GitHub.
2. ارفع الملفات: `index.html` و`styles.css` و`script.js`.
3. افتح Settings ثم Pages.
4. اختر Deploy from a branch.
5. اختر فرع `main` والمجلد `/root` ثم Save.

سيظهر رابط الموقع خلال دقائق.
